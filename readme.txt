== Description ==

To use this plugin, first install dependencies with:
<pre>bower install</pre>

<h3>Notes</h3>
<ul>
<li>Only one carousel can be used per page</li>
<li>For each sidebar used, ensure your theme includes a page-template that includes a sidebar with the call "dynamic_sidebar('austeve_gallery_X');", where 'X' is a number starting at 1</li> 
</ul>
