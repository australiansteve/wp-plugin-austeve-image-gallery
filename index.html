AUSteve Image Gallery

WordPress plugin to display a slick.js carousel of images on a page or post

Introduction

The AUSteve Image Gallery plugin is my preferred way to insert image galleries into my WordPress pages and posts.

It is currently only supported for themes that use Zurb Foundation, which all of my custom client themes do.

Installation

Administration Page

Once installed, a link to the administration page appears under the Settings menu - 'AUSteve Gallery settings'. Settings are as follows

Number of sidebars

Number of sidebars determines how many different sets of galleries you can preview on your website. For example you might have a set of Residential projects, you'll need one sidebar for that. Then you might have Commercial projects, you'll need another for that. Setting this number too high shouldn't have a detrimental effect, but try to set it to the minimum number required for sanity's sake.

Gallery preview format

Sometimes, a theme might be configured to handle more than one 'type' of preview page. If this is the case, use this setting to change the way preview pages are laid out. If you are not sure if your theme supports multiple formats, it's best not to change this setting. If you want to display your preview pages a different way, contact Steve to discuss your options.
